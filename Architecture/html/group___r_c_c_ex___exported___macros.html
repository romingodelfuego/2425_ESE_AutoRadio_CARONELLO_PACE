◆ __HAL_RCC_GET_ADC_SOURCE

#define __HAL_RCC_GET_ADC_SOURCE ( )
Value:
((__HAL_RCC_ADC_IS_CLK_ENABLED() != 0U) ? RCC_ADCCLKSOURCE_SYSCLK : RCC_ADCCLKSOURCE_NONE)

Macro to get the ADC clock source.

Return values
Theclock source can be one of the following values:
  • RCC_ADCCLKSOURCE_NONE No clock selected as ADC clock
  • RCC_ADCCLKSOURCE_SYSCLK System Clock selected as ADC clock
